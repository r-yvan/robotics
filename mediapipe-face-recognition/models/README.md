# Face Recognition System - Models Directory

This directory contains the trained LBPH face recognition model and label mapping.

## Files (Generated after training)

- **lbph_face_recognizer.yml** - Trained LBPH model
- **label_map.json** - Mapping of label IDs to person names

## How to Generate

Run the training script after capturing face images:

```bash
python 2_train_model.py
```

## Note

This directory is kept in the repository for assignment submission, but the actual model files will be generated when you run the training script.
